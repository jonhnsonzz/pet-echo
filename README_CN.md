# PetEcho 宠物异常助手 🐾

**你的宠物不对劲？先问 PetEcho。**

> 凌晨2点，猫拉稀了。你不知道该立刻去医院，还是观察一天再说。上网搜，全是宠物医院的广告。PetEcho 告诉你：这种情况常见原因是什么、什么时候该就医、什么时候可以再观察。

## 😰 你是不是遇到过这些情况

- 宠物拉稀/呕吐了，但精神还行——要不要去医院？
- 猫狗突然不爱吃饭——是挑食还是真的不舒服？
- 宠物行为异常（乱叫/躲起来/不动）——是心情不好还是生病了？
- 半夜宠物出状况，宠物医院关门了——你能做点什么？
- 想去医院但又怕被宰，想观察但怕拖严重了

**PetEcho 就是为这些时刻设计的。**

## 💡 PetEcho 能做什么

在你描述宠物的情况后，PetEcho 会：

1. **判断严重程度** — 这种情况常见吗？是小问题还是需要立刻就医
2. **给出观察建议** — 在家可以做什么、观察几天、什么症状出现必须就医
3. **记录变化趋势** — 持续追踪宠物的异常情况，有数据才能更好判断
4. **减少不必要的焦虑** — 不是所有异常都需要半夜冲医院，但你不确定，PetEcho 帮你判断

## 📱 在什么设备上使用

手机浏览器打开即可使用（微信、Chrome、Safari 均支持）。
不需要下载 App，不需要注册账号。

## 🔒 数据隐私

- 数据只存在你的设备本地，不上传到任何服务器
- 不需要手机号、邮箱、账号
- 不追踪、不贩卖数据

## 🚀 快速开始

### 环境要求
- Python 3.8+
- OpenAI API Key

### 安装

```bash
# 克隆仓库
git clone https://github.com/jonhnsonzz/pet-echo.git
cd pet-echo

# 安装依赖
pip install -r requirements.txt

# 配置 API Key
cp .env.example .env
# 编辑 .env 文件，填入你的 OpenAI API Key
# OPENAI_API_KEY=你的key
```

### 运行

```bash
python app.py
```

然后用手机浏览器打开 `http://你的服务器IP:5000`

## 💡 产品理念

宠物主人是最好的观察者——你每天都和宠物在一起，你最了解它正常的样子。

但你缺少一个工具，把你的观察和医学判断之间搭起一座桥。

PetEcho 不是医疗器械，不能替代兽医。它做的是：**在你犹豫要不要去医院的时候，帮你做一个更有信息的判断**。
